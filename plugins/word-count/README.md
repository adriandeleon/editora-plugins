# Word Count plugin

A live stats **tool window** (docks right): lines, words, characters (with/without spaces), and estimated
reading time for the active buffer — refreshed on a ~1.5s timer and via a Refresh button (skips buffers
over ~2M chars). Pure stats in `WordStats`. Build with `./build.sh`; install via *Settings → Plugins →
Install from file…*. See [`../../docs/plugins.md`](../../docs/plugins.md).
