# Color Picker plugin

A color-picker **tool window** (docks right): choose a color and an output format (HEX / `rgb()` / `rgba()`),
then **Insert at caret**. Uses JavaFX `ColorPicker` + `insertAtCaret`. Build with `./build.sh`; install via
*Settings → Plugins → Install from file…*. See [`../../docs/plugins.md`](../../docs/plugins.md).
