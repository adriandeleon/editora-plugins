# Calculator plugin

A calculator **tool window** (docks right): type an arithmetic expression (`+ - * / % ^`, parentheses,
unary minus), see the result live, and **Insert result** at the caret (Enter inserts). Pure recursive-descent
evaluator in `Calc`. Build with `./build.sh`; install via *Settings → Plugins → Install from file…*. See
[`../../docs/plugins.md`](../../docs/plugins.md).
