# Box Banner plugin

`Banner: Box` — wraps the selection (or whole document) in an ASCII `*` box, padding lines to the widest:

```
**********
* hello  *
* world! *
**********
```

Pure logic in `BannerText`. Build with `./build.sh`; install via *Settings → Plugins → Install from file…*.
See [`../../docs/plugins.md`](../../docs/plugins.md).
