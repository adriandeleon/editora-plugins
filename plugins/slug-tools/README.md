# Slug & Sequence plugin

`Slug: Slugify` (URL-safe slug of the selection) · `Sequence: Number Lines` (prefix `1. `, `2. `…) ·
`Sequence: Fill 1..N` (replace selected lines with their index). Operates on the selection, else the whole
document. Pure logic in `SlugSeq`. Build with `./build.sh`; install via *Settings → Plugins → Install from
file…*. See [`../../docs/plugins.md`](../../docs/plugins.md).
