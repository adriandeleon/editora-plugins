# Reveal & Terminal plugin

`Reveal in File Manager` — reveals the active file in the OS file manager (macOS Finder / Windows Explorer /
Linux `xdg-open`). `Open Terminal Here` — opens a terminal at the file's folder (macOS Terminal / Windows
`cmd` / Linux `x-terminal-emulator`). Per-OS commands spawned via `ProcessBuilder`. Build with `./build.sh`;
install via *Settings → Plugins → Install from file…*. See [`../../docs/plugins.md`](../../docs/plugins.md).
